# RF-TAL-02 El Sistema debe permitir consultar y actualizar el registro de un tallerista

## Descripción

El Sistema debe permitir **consultar** y **actualizar** un **registro de tallerista** existente, tanto sus datos de contacto como la información de su taller propuesto. Aplica después de que el registro fue creado en `RF-TAL-01`, durante la **revisión** o el ajuste de la propuesta. El resultado es el mismo registro con su información vigente, reflejando los cambios solicitados.
