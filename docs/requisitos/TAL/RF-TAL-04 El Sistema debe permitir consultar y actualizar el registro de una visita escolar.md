# RF-TAL-04 El Sistema debe permitir consultar y actualizar el registro de una visita escolar

## Descripción

El Sistema debe permitir **consultar** y **actualizar** un **registro de visita escolar** existente, tanto los datos de contacto de la institución como la información de la visita propuesta. Aplica después de que la solicitud fue creada en `RF-TAL-03`, durante su **revisión** o ajuste. El resultado es el mismo registro con su información vigente, reflejando los cambios solicitados.
