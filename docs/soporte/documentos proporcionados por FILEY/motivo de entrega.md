# Motivo de entrega

En el presente correo te mando adjuntos los documentos que mis compañeros Hipólito, Elvira y Manuel han preparado para generar el sistema de registro de las actividades de la FILEY.

En el documento anexo que se llama **"Detalles para el registro en línea"** están las recomendaciones de mi compañero Jesús Hipólito para la construcción del sistema de registro, plazos adecuados y otras precisiones.

En años anteriores hemos abierto las convocatorias desde el mes de julio debido a que el trabajo manual llevaba más tiempo.

Si ahora se sistematiza permitirá acortar los plazos y agilizar el proceso. Por ello ahora se propone el **10 de agosto** como la fecha de apertura de las convocatorias.

Estaría pendiente aún para integrar al sistema general el modo de registro de las siguientes actividades:

- Escénicas (alrededor de 50 actividades)
- Cinematográficas (alrededor de 20 actividades)

Como hablamos en la reunión, las actividades de Artes Visuales, Cinematográficas y Escénicas no tienen convocatoria abierta, de manera que su registro sería de manera interna por nuestro personal. En el caso de Artes Visuales como tienen particularidades sí te adjunto el Excel modelo de los rubros que se suelen registrar por actividad.

En el caso de Talleres y visitas escolares tiene ya un sistema de convocatoria. La que puede servir de muestra se encuentra en el siguiente enlace de Forms:

<https://docs.google.com/forms/d/e/1FAIpQLSdw1oQExa86_64I_B9Kfu38XUeiSwnJxFTooVmIRfVNpROfMA/viewform?usp=publish-editor>

Asimismo, en el correo se adjuntan:

- Excel con el ejemplo de la base de datos del registro de propuestas
- Convocatoria general de actividades
- Excel del programa maestro (Calendario) con el detalle por sala. El modelo ya contempla el calendario 2027 con algunos contenidos 2026 de muestra para que sirvan de ejemplo.
- Word en PDF ejemplo con el programa general de todas las actividades (sirve para construir el programa impreso)
- El modelo del registro de propuestas literarias y académicas que hemos utilizado y que ahora puede servir para alimentar el sistema.
- Un word explicativo sobre las necesidades específicas, no solo para el sistema del registro de los talleres sino también para la asignación de las escuelas a los talleres ya aceptados y agendados.
- Las notas con observaciones complementarias al registro de talleres.
- El Excel con la información correspondiente a las actividades de artes visuales que tienen características propias y los datos que habría que incluir para su registro.

Muchas gracias por tu atención y estamos en comunicación para resolver las dudas que se presenten durante el proceso.
