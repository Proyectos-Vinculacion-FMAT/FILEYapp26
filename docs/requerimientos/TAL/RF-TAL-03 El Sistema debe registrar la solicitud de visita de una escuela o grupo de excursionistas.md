# RF-TAL-03 El Sistema debe registrar la solicitud de visita de una escuela o grupo de excursionistas

## Descripción

El Sistema debe registrar la **solicitud de visita** de una **escuela** o **grupo de excursionistas** que desea asistir a la feria con un grupo de niños o jóvenes. Captura los **datos de contacto** de la institución y la información de la **visita propuesta**. Aplica en la etapa de convocatoria, previa a la feria, cuando la escuela presenta su solicitud. El resultado es un **registro de visita escolar** disponible para su consulta y coordinación.
